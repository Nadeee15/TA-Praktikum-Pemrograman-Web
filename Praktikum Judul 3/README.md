# StarkCorp — Landing Page Responsif  

Landing page responsif untuk **perusahaan teknologi fiktif** bernama **StarkCorp**, yang berfokus pada inovasi perangkat lunak dan layanan cloud.  
Dibangun menggunakan **Tailwind CSS** untuk tampilan modern, bersih, dan fleksibel di berbagai perangkat.

## Struktur Halaman
Website terdiri atas enam komponen utama:
1. Header / Navigation  
2. Hero Section  
3. Features Section  
4. About Section  
5. Contact Form  
6. Footer  

---

## Teknologi yang Digunakan
- **HTML**  
- **Tailwind CSS (via CDN)**  
- **Remix Icon / Lucide Icons** untuk ikon modern  
- Tidak menggunakan framework JavaScript tambahan
